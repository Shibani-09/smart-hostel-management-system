# 🌌 Vision

The Smart Hostel Management System is a modern full-stack web platform engineered to simplify and digitize hostel operations through automation, security, and intelligent management tools.

Designed with enterprise-level architecture and futuristic SaaS-inspired UI principles, the platform enables administrators, wardens, and students to interact inside one connected digital ecosystem.

This project combines powerful backend systems with immersive frontend experiences to create a next-generation smart campus infrastructure.

# ⚙️ Quick Installation Guide

# 🔹 Backend Setup

## 1️⃣ Navigate to Backend

```bash
cd backend
```

## 2️⃣ Install Dependencies

```bash
npm install
```

## 3️⃣ Configure Environment Variables

Copy:

```bash
backend/.env.example
```

To:

```bash
backend/.env
```

Then add your MongoDB URI, JWT secret, and server configuration.

## 4️⃣ Start Backend Server

```bash
npm run dev
```

---

# 🔹 Frontend Setup

## 1️⃣ Navigate to Frontend

```bash
cd frontend
```

## 2️⃣ Install Dependencies

```bash
npm install
```

## 3️⃣ Configure Frontend Environment

Copy:

```bash
frontend/.env.example
```

To:

```bash
frontend/.env
```

Set:

```bash
REACT_APP_API_URL
```

## 4️⃣ Start Frontend Application

```bash
npm start
```
